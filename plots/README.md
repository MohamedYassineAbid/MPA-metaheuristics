# Plots Directory

Place your CEC2017 function visualization images here with the following names:

- `f2.png` - CEC2017 Function F2 visualization
- `f4.png` - CEC2017 Function F4 visualization
- `f12.png` - CEC2017 Function F12 visualization
- `f25.png` - CEC2017 Function F25 visualization

These images will be displayed in the "Static Plots" tab of the Streamlit application.

## Image Requirements

- Format: PNG (recommended) or JPG
- Resolution: Minimum 800x600 pixels
- Content: 2D or 3D visualization of the function landscape
